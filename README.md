# Water.DRIE

Water-quality prediction prototype with Streamlit experiments, preprocessing scripts, and time-series modeling.

## What It Does

- Preprocesses water-quality style data.
- Experiments with time-series and ML prediction scripts.
- Includes Streamlit-style app experiments for interactive use.

## Tech

Python, Streamlit, time-series modeling, data preprocessing, ML prediction.

## Repository Map

- `data_preprocessing.py`, `model_training.py` - ML pipeline pieces.
- `time-seriesmodel.py`, `tsmodel.py` - time-series experiments.
- `app.py`, `main*.py` - app and execution experiments.

## Status

Prototype. The repository captures exploration toward a usable water-quality forecasting tool.
